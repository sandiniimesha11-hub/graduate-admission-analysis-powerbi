# Graduate Admission Analysis Dashboard using Power BI 

## Project Overview

This project presents an interactive Power BI dashboard developed to analyze graduate admission trends and understand how academic and profile-related factors influence admission probability.

The dashboard provides insights into the relationship between CGPA, GRE score, TOEFL score, university rating, research experience, SOP, and LOR scores.

## Objectives
• Analyze factors affecting graduate admission probability

• Identify relationships between academic performance and admission outcomes

• Create interactive dashboards for data-driven decision making

• Generate actionable insights for prospective students

## Dataset

Dataset: Graduate Admission Dataset

Source: Kaggle

Records: 500 student applications

Features:

• GRE Score

• TOEFL Score

• University Rating

• SOP Score

• LOR Score

• CGPA

• Research Experience

• Admission Probability
